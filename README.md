# hello-world

这是一个给新手用的「灵感记录网页」示例。

## 30 秒预览（最简单，只做这 3 步）

1. 在项目里找到 `index.html`。
2. 双击它。
3. 浏览器会直接打开页面（这就是界面）。

如果双击没反应：
- 右键 `index.html` → 选择“打开方式” → Chrome / Edge。

---

## 你现在“只看到代码”怎么办？

你需要把代码当成一个网页文件来打开：

- 如果你是在聊天窗口里看到代码：
  1. 新建一个文本文件，命名为 `index.html`
  2. 把代码完整复制进去并保存
  3. 双击 `index.html`，就能看到界面

- 如果你已经在项目文件夹里有 `index.html`：
  - 直接双击就行，不需要运行任何命令

---

## 备用方法（VS Code）

1. 用 VS Code 打开项目。
2. 安装 **Live Server** 扩展。
3. 右键 `index.html` → **Open with Live Server**。
4. 浏览器会打开页面（例如 `http://127.0.0.1:5500/index.html`）。

---

## 本地存储到底存到哪里？

这个页面把内容存到 **浏览器的 localStorage**，键名是：`inspiration-note-v1`。

- 它**不在项目文件夹里**，也**不是一个你能直接看到的 txt 文件**。
- 它在浏览器内部的数据目录中（按“浏览器 + 设备 + 用户配置”隔离）。
- 同一个浏览器里刷新页面会保留；换浏览器、换电脑通常看不到原内容。

## 如何亲眼看到它（推荐）

1. 打开网页后，按 `F12`（开发者工具）。
2. 找到 `Application`（或 `存储`）标签。
3. 在左侧选 `Local Storage`。
4. 点开当前页面地址，查看键 `inspiration-note-v1` 的值。
